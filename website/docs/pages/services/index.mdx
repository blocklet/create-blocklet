Blocklet Services is a set of convenient APIs for developers to call Blocklet Server's functions.

By using Blocklet Services, a Blocklet application can quickly integrate login authentication service, message notification, etc.

Currently supported service capabilities:

- [Auth Service](/services/auth-service)
- [Notification Service](/services/notification-service)
