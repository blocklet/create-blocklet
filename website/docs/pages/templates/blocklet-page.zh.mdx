## 使用模板

```shell
npm init blocklet@latest my-blocklet blocklet-page
```

## 开发

:::Alert
前提条件: 

- [Blocklet Server](https://developer.blocklet.io/docs/zh/quick-start/blocklet-server)
- [Node.js 14+](https://developer.blocklet.io/docs/zh/quick-start/nodejs)
:::

```shell
# install dependencies
npm install
# run the blocklet in development environment mode
blocklet dev
```

该应用基于 xmark 构建，你可以尝试编辑 pages 目录中的页面，或者通过 .blocklet-pages/config.yml 定制你的站点。进一步了解请参考 [Blocklet Pages](https://pages.blocklet.io/)。
