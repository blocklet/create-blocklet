## 使用模板

```shell
npm init blocklet@latest my-blocklet vue2-dapp
```

## 开发

:::Alert
前提条件: 

- Blocklet Server
- Node.js 14+
:::

```shell
# install dependencies
npm install
# run the blocklet in development environment mode
blocklet dev
```

应用代码主要存放于以下几个目录中:  

- api 目录: 存放服务端接口模块
- src 目录: 存放 Vue 前端应用相关模块

你可以尝试编辑目录中的模块，然后刷新页面查看修改后的效果。

## 技术栈

- Vue 2
- @vue/cli
- axios
- Express
