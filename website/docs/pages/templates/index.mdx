Thanks to Blocklet Server's powerful compatibility capabilities, we can easily make existing framework templates run on Blocklet Server with minimal modification costs.

We currently support the following templates.

- [Blocklet Page](./blocklet-page)
- [Doc Site](./doc-site)
- [Express API](./express-api)
- [Html Static](./html-static)
- [Nextjs Dapp](./nextjs-dapp)
- [React Dapp](./react-dapp)
- [React Static](./react-static)
- [Solidjs Dapp](./solidjs-dapp)
- [Solidjs Static](./solidjs-static)
- [Svelte Dapp](./svelte-dapp)
- [Svelte Static](./svelte-static)
- [Vue Dapp](./vue-dapp)
- [Vue Static](./vue-static)
- [Vue2 Dapp](./vue2-dapp)
- [Vue2 Static](./vue2-static)
