## Usage

```shell
npm init blocklet@latest --template=express-api my-blocklet
```

## Development

:::Alert
Prerequisites: 

- [Blocklet Server](https://developer.blocklet.io/docs/en/quick-start/blocklet-server)
- [Node.js 14+](https://developer.blocklet.io/docs/en/quick-start/nodejs)
:::

```shell
# install dependencies
npm install
# run the blocklet in development environment mode
blocklet dev
```

This is the most basic API application, with only one interface. You can modify the interface logic or add a new interface by editing the api/index.js module.
