## 使用模板

```shell
npm init blocklet@latest my-blocklet express-api
```

## 开发

:::Alert
需要准备 Blocklet Server 环境.
:::

```shell
# install dependencies
npm install
# run the blocklet in development environment mode
blocklet dev
```

这是一个最简单的 API 应用，只提供了一个接口，你可以编辑 api/index.js 模块，尝试修改接口逻辑或增加新的接口。
