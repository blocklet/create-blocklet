## 使用模板

```shell
npm init blocklet@latest my-blocklet solidjs-static
```

## 开发

:::Alert
需要准备 Blocklet Server 环境.
:::

```shell
# install dependencies
npm install
# run the blocklet in development environment mode
blocklet dev
```

应用代码主要存放于 src 目录中，你可以尝试编辑目录中的模块，然后刷新页面查看修改后的效果。

## 技术栈

- Solidjs
- Vite
