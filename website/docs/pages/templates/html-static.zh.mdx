## 使用模板

```shell
npm init blocklet@latest my-blocklet html-static
```

## 开发

:::Alert
前提条件: 

- Blocklet Server
:::

```shell
# run the blocklet in development environment mode
blocklet dev
```

网页程序代码存放于 app 目录中，包括 logo、css 样式、HTML 等文件，你可以尝试编辑 HTML 和 css 文件，然后刷新页面查看修改后的效果。
