## Usage

```shell
npm init blocklet@latest my-blocklet vue-static
```

## Development

:::Alert
Prerequisites: 

- Blocklet Server
- Node.js 14+
:::

```shell
# install dependencies
npm install
# run the blocklet in development environment mode
blocklet dev
```

应用代码主要存放于 src 目录中，你可以尝试编辑目录中的模块，然后刷新页面查看修改后的效果。

- Vue 3
- Vite
