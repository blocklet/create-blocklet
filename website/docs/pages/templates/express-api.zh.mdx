## 使用模板

```shell
npm init blocklet@latest my-blocklet express-api
```

## 开发

:::Alert
前提条件: 

- Blocklet Server
- Node.js 14+
:::

```shell
# install dependencies
npm install
# run the blocklet in development environment mode
blocklet dev
```

这是一个最简单的 API 应用，只提供了一个接口，你可以尝试编辑 api/index.js 模块，修改接口逻辑或增加新的接口。
