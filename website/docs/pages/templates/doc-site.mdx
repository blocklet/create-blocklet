## 使用模板

```shell
npm init blocklet@latest my-blocklet doc-site
```

## 开发

:::Alert
需要准备 Blocklet Server 环境.
:::

```shell
# install dependencies
npm install
# run the blocklet in development environment mode
blocklet dev
```

该应用基于 xmark 构建，你可以编辑 pages 目录中的页面，或者通过 .blocklet-pages/config.yml 定制你的站点。进一步了解请参考 [Blocklet Pages](https://pages.blocklet.io/)。
