## Usage

```shell
npm init blocklet@latest --template=docsite my-blocklet
```

## Development

:::Alert
Prerequisites:

- [Blocklet Server](https://developer.blocklet.io/docs/en/quick-start/blocklet-server)
- [Node.js 14+](https://developer.blocklet.io/docs/en/quick-start/nodejs)
:::

```shell
# install dependencies
npm install
# run the blocklet in development environment mode
blocklet dev
```

The application is built on Blocklet Pages, and you can try to edit pages in the pages directory or customize your site with .blocklet-pages/config.yml. See [Blocklet Pages](https://pages.blocklet.io/) for more information.
