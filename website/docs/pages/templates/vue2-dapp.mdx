## Usage

```shell
npm init blocklet@latest my-blocklet vue2-dapp
```

## Development

:::Alert
Prerequisites: 

- [Blocklet Server](https://developer.blocklet.io/docs/en/quick-start/blocklet-server)
- [Node.js 14+](https://developer.blocklet.io/docs/en/quick-start/nodejs)
:::

```shell
# install dependencies
npm install
# run the blocklet in development environment mode
blocklet dev
```

应用代码主要存放于以下几个目录中:  

- api 目录: 存放服务端接口模块
- src 目录: 存放 Vue 前端应用相关模块

你可以尝试编辑目录中的模块，然后刷新页面查看修改后的效果。

## Tech Stack

- Vue 2
- @vue/cli
- axios
- Express
