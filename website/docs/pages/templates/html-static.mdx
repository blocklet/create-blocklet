## Usage

```shell
npm init blocklet@latest my-blocklet html-static
```

## Development

:::Alert
Prerequisites: 

- [Blocklet Server](https://developer.blocklet.io/docs/en/quick-start/blocklet-server)
:::

```shell
# run the blocklet in development environment mode
blocklet dev
```

The app directory contains the source code of web page, including logo, CSS style, HTML, and other files. You can try editing the HTML and CSS files, then refreshing your browser to see your changes.
