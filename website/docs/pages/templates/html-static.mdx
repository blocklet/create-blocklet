该模板是一个最简单的 HTML 静态页面应用。可以用于将已有的静态网页程序包装成 Blocklet。

## 开发

:::Alert
需要准备 Blocklet Server 环境.
:::

```shell
# install dependencies
npm install
# run the blocklet in development environment mode
blocklet dev
```

你可以尝试编辑 app 目录中的 html 和 css 文件，然后刷新页面查看修改后的效果。
