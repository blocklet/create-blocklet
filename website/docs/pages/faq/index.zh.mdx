## 文档是如何组织的？

对该文档的组织方式的高层次概述将帮助你知道在哪里寻找某些东西。

- [快速入门](/docs/zh/quick-start)带领你通过一系列的步骤来创建一个网站。如果你是 Blocklet 新手，可以从这里开始。
- [参考指南](/docs/zh/reference/blocklet-structure) 包含创建 Blocklet 各个方面的技术参考。它们描述了它是如何工作的以及如何使用它，但假定你对关键概念有基本的了解。
- [操作指南](/docs/zh/how-to/create-single-blocklet) 是教程，它们指导你完成解决关键问题和用例的步骤。

## 未找到 global / buffer / process 对象的问题

在 vite-plugin-blocklet 中提供了解决方案，通过传入参数 `disableNodePolyfills: false` 即可解决该问题。

完整示例：
``` js
import { defineConfig } from 'vite';
import react from '@vitejs/plugin-react';
import { createBlockletPlugin } from 'vite-plugin-blocklet';
import svgr from 'vite-plugin-svgr';

export default defineConfig((s) => {
  return {
    plugins: [react(), createBlockletPlugin({ disableNodePolyfills: false }), svgr()],
  };
});

```

## 为什么 GitHub Actions 没有触发？

如果你使用 GitHub Actions 来实现 Blocklet 打包和发布工作流，请确保 `.github/workflows/main.yml` 的分支和你代码仓库的默认分支一致，否则 GitHub Actions 可能不会被触发。
