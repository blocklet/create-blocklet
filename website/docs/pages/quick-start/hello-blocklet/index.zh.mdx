Create Blocklet 是一个官方支持的 Blocklet 创建工具，可以快速创建 Blocklet ，并且可以自动添加到项目中。

## 环境准备

首先得有 [node](https://nodejs.org/en/)，并确保 node 版本是 16.0 或以上。（mac 下推荐使用 [nvm](https://github.com/creationix/nvm) 来管理 node 版本）

## 脚手架

通过官方工具创建项目

```shell
yarn create blocklet my-blocklet
# 或 npm create blocklet my-blocklet
# 或 pnpm create blocklet my-blocklet
```

通过 `@blocklet/cli` 工具创建项目

```shell
blocklet create my-blocklet
```

## 启动项目

```shell
yarn dev
```

打开控制台输出的网址，就能看到当前启动的项目了。

## 部署发布

### 构建

```shell
yarn bundle
```

将应用构建为可以部署到 Blocklet Server 中的代码包，或可上传至 Blocklet Store 的应用包。构建产物默认生成到 `.blocklet` 下

### 部署

目前的部署支持两种形式：

1. 直接通过 deploy 命令来部署
2. 通过上传到 Blocklet Store 中进行部署

在发布到 Blocklet Store 之前，可以通过部署到本地的 Blocklet Server 来做本地验证

```shell
yarn deploy
```

运行后会在本地的 Blocklet Server 中的基石程序列表中看到刚刚部署的应用

### 上传

```shell
yarn upload
```

你可以使用这个命令将 blocklet 应用上传到 Blocklet Store 中，详细的文档 [在这里](https://developer.blocklet.io/docs/zh/store)
