# Welcome to <strong className="color-#2482F6">Create Blocklet</strong>, your <strong className="color-red">Blocklet Project Assistant</strong>

Create Blocklet is a command line tool that makes it smooth for developers to start their blocklet projects. With Create Blocklet you can bootstrap a blocklet project that contains best practices from ArcBlock team within minutes. Start your blocklet today, for fun and profit!
