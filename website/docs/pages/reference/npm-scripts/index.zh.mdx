通过 `create-blocklet` 工具创建出来的项目，会在 `package.json` 中配置好对应的 `npm script`来方便开发者使用。

`npm script` 理论上可以使用 `npm`、`yarn`、`pnpm` 命令执行执行。

## dev

开发模式下运行 `Blocklet` 项目（此命令需要开发环境中有安装 `@blocklet/cli` 工具），运行命令后，稍等片刻，终端中会打印一个 URL ，你可以通过这个 URL 查看当前运行的项目。

## bundle

将项目打包为可以部署到 `Blocklet Server` 的应用包。打包后的产物是压缩的，文件名包含了哈希值。
至此，你的 `Blocklet` 项目已经可以部署到 `Blocklet Server` 中。

## deploy

通过这个命令，可以轻松将你打包的 `Blocklet` 应用部署到你本地的 `Blocklet Server` 中（确保你的 `Blocklet Server` 当前处于运行状态中）。

## upload

将项目上传至 `Blocklet Store` 中。可以是官方的 `Blocklet Store`，也可以是你自己的 `Blocklet Store`。只要你是 `Blocklet Store` 的开发者用户，你可以通过这个命令将项目上传至你所拥有权限的 `Blocklet Store` 中。

## lint

使用 `Eslint` 对你的代码进行检查。

## lint\:fix

使用 `Eslint` 快速修复你的代码。
