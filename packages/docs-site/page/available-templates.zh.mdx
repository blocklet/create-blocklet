得益于 Blocklet Server 强大的兼容能力，我们可以轻松的将现有的框架模板通过极小的改造成本使其能够运行在 Blocklet Server 上。

目前我们已经支持了以下模板：

- Blocklet Page
- Express
- Plain HTML
- Next.js
- React (with craco)
- React + GUN (with craco)
- React + Express (with craco)
- Vue2 (with vue-cli)
- Vue2 + Express (with vue-cli)
- Vue3 (with vite)
- Vue3 + Express (with vite)
- SolidJS (with vite)
- SolidJS + Express (with vite)
- Svelte (with vite)
- Svelte + Express (with vite)

更多的模板正在积极筹备中...
